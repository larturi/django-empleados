# Empleados App
App de Empleados con Django

# Get Started:

Instalar dependencias:
```
pipenv install
```

Crear migraciones:
```
python3 manage.py makemigrations
python3 manage.py migrate
```

Ejecutar servidor local
```
python manage.py runserver --settings=empleados.settings.local
```
